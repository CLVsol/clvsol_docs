Samba installation
==================

References:

* `Installing Samba Server in Mint 18 <https://forums.linuxmint.com/viewtopic.php?t=224391>`_

To install `Samba <https://www.samba.org/>`_, execute the followind:

#. **Install Samba files**

	First thing we need to do is to install samba, go to Software center in Ubuntu and search for samba then install the package. If you want to install it via terminal then copy this command::

		sudo apt-get install samba --install-recommends
