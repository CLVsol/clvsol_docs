OpenERP Client Library installation
===================================

References:

* `openerp-client-lib 1.1.2 <https://pypi.python.org/pypi/openerp-client-lib/1.1.2>`_
* `OpenERP Client Library <https://github.com/OpenERP/openerp-client-lib>`_
* `Ubuntu: how to install easy_install <http://www.hacksparrow.com/ubuntu-how-to-install-easy_install.html>`_

To install `OpenERP Client Library <https://github.com/OpenERP/openerp-client-lib>`_, execute the following commands
	::

		sudo apt-get install python-setuptools
		sudo easy_install openerp-client-lib
