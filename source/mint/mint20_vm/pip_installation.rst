================
pip installation
================

References:

* `pip <https://pip.pypa.io/en/stable/>`_
* `How to install pip on Linux <http://ask.xmodulo.com/install-pip-linux.html/>`_

:red:`(Not Executed)` To install `pip <https://pip.pypa.io/en/stable/>`_, execute the followind command
	::

		sudo apt-get install python-pip 

To install `pip3 <https://pip.pypa.io/en/stable/>`_, execute the followind command
	::

		sudo apt install python3-pip
