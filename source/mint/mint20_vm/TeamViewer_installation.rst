=======================
TeamViewer installation
=======================

References:

	* `How To Install TeamViewer on Linux Mint 20 / Linux Mint 19 <https://www.itzgeek.com/how-tos/linux/linux-mint-how-tos/how-to-install-teamviewer-on-linux-mint-19-linux-mint-18.html>`_

	#. Install TeamViewer:

			::

				sudo apt update

				wget https://download.teamviewer.com/download/linux/teamviewer_amd64.deb

				sudo apt install -y ./teamviewer_amd64.deb

	#. Start TeamViewer in Cinnamon desktop: Menu >> Internet >> TeamViewer.
