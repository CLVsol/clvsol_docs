=====================
LibreCAD Installation
=====================

References:

* `LibreCAD <https://librecad.org/>`_
* `How to install LibreCAD 2.2.0 in Linux Mint 17.3 <https://www.linuxhelp.com/how-to-install-librecad-2-2-0-in-linux-mint-17-3>`_

#. install LibreCAD in Linux Mint:

    #. To start the installation procedure, first add the LibreCAD package into repository via ppa:

        ::

            sudo add-apt-repository ppa:librecad-dev/librecad-daily

    #. Now update the target machine using the apt-get update command:

        ::

            sudo apt-get update

    #. Install the LibreCAD package by executing the following command:

        ::

            sudo apt-get install librecad

    #. Launch the LibreCAD application by typing the application name from Linux Mint dashboard and run the application.
