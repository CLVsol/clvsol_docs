===========
SublimeREPL
===========

References:

* `SublimeREPL <https://github.com/wuub/SublimeREPL>`_

**Installation:**

    To install SublimeREPL trough Package Control show the Command Palette (Tools -> Command Palette) and write "install package", then write "SublimeREPL" in the input window popup that just appeared (that is the SublimeText 3 Command Palette) and press "enter".
