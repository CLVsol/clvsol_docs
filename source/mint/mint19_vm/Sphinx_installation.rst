===================
Sphinx installation
===================

#. To install `Sphinx <http://sphinx-doc.org/>`_, execute the followind command

	::

		sudo apt-get install python-sphinx

#. Para atualização do Sphinx/Alabaster

	::

		sudo pip install -U sphinx

#. Para atualização do Sphinx/Alabaster (usando pip3)

	::

		sudo pip3 install -U sphinx

#. Para instalação do "sphinx_rtd_theme" (usando pip3)

	::

		sudo pip3 install sphinx_rtd_theme
