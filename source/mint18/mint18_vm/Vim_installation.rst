================
Vim installation
================

References:

* `Vim <http://www.vim.org/>`_
* `How to install vim editor in Linux Mint <http://sharadchhetri.com/2014/01/22/how-to-install-vim-editor-in-linux-mint/>`_

To install `Vim <http://www.vim.org/>`_, execute the followind command
	::

		sudo apt-get install vim

Example of use
