Check for Any Last Minute Updates
=================================

Select an update policy:

 #. Click on Menu - Administration - Update Manager
 #. Select an update policy: the second one, named "Optimize stability and security!"

How to check for updates on Linux Mint:

 #. Launch the ‘Update Manager’
 #. Click the refresh to check for updates
 #. Install (if needed)

Using the command console, execute the followind commands

	::

		sudo apt-get update
		sudo apt-get -y upgrade
